# WAKE WORD SETUP

ALICE does not come with wake word models by default. This is for a variety of reasons, but the primary ones are twofold:

1.  The tools used to generate them are a complicated mish-mash of licenses that make it fine to train them for personal
    use, but distributing them is sketchy at best.
2.  You probably want to change the wake word anyway. The whole point of this assistant is the customizable personality
    and identity.

This file exists to tell you how to train your own custom wake word model, or train your own default "Hey Alice"
wake word, if you happen to _like_ the unmitigated ALICE "experience". I won't judge.
